# Care Intake Console Web

Angular frontend for a synthetic outpatient intake console. It connects to `care-intake-console-api` and covers queue management, patient and appointment flows, intake capture, triage review, and follow-up tracking.

## Stack

- Angular 18
- Signals
- Tailwind
- ApexCharts

## Run

```bash
npm install
npm start
```

By default the app expects the API at `http://127.0.0.1:3000`.
